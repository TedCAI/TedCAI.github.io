2.Abstract Data Type vs Data Structure
(1)ADT 		 = 数据模型 + 定义在该模型上额度一组操作
			   (抽象定义,外部逻辑特性,不考虑复杂度,不涉及数据储存方式)
(2)数据结构  = 基于某种特定语言,实现ADT的一整套算法
			   (具体实现,完整算法,考虑复杂度,考虑数据储存机制)
